# AuraSVN

A Flutter-based desktop application that brings intelligence to SVN repository visualization and analysis.

## Features

- **AI-Powered Analysis** - Leverages AI to analyze SVN repository structure and branch relationships
- **Markdown Branch Documentation** - Automatically generates Markdown notes for each branch, capturing topology and log data
- **Infinite Canvas Branch Viewer** - Interact with an infinite, pannable canvas to explore and visualize branch relationships with zoom and focus

## Getting Started

```bash
flutter pub get
flutter run
```

## Tech Stack

- Flutter (Desktop)
- Python backend scripts for SVN processing
- Graph visualization engine
